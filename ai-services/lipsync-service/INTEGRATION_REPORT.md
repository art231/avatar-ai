#### 4. Использование улучшенного процессора:

```python
# Проверка доступности моделей
print(f"Модели доступны: {processor.models_available}")

# Обработка видео
success = processor.process_video(
    video_path="input.mp4",
    audio_path="audio.wav",
    output_path="output.mp4",
    quality="high",
    sync_accuracy=0.9
)
```
